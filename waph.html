Course Overview

Web Application Programming and Hacking is a course focused on web development, web services, APIs, GitHub deployment, and web application security. Through labs, hackathons, and projects, I practiced both building and securing web applications.

Hands-on Labs and Projects

  • Lab 0: Development environment setup, GitHub, SSH, and basic tools.
  • Lab 1: Foundations of the Web using HTML, forms, JavaScript, and CGI.
  • Lab 2: Front-end web development using HTML, CSS, JavaScript, Ajax, jQuery, and APIs.
  • Hackathon 1: Cross-site scripting attacks and defenses.
  • Individual Project 1: Professional profile website deployed on GitHub Pages with API integration.

What I Learned

I learned how to create front-end webpages, use JavaScript for interactivity, integrate public APIs, deploy a website on GitHub Pages, and understand basic web security concepts.
