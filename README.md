# Technical Requirements

## Digital Clock

A digital clock was implemented using JavaScript and Moment.js. The clock updates every second and displays the current date and time.

## Analog Clock

An analog clock was implemented using the HTML5 Canvas element and JavaScript drawing functions.

## Show/Hide Email

A Show/Hide Email button was implemented using jQuery to display or hide contact information.

## Additional JavaScript Feature

A Dark Mode toggle button was implemented to allow users to switch between light and dark themes.

---

# Public API Integration

## JokeAPI

The JokeAPI was integrated to display a random joke.

API URL:

https://v2.jokeapi.dev/joke/Any

The joke automatically refreshes every minute.

## Dog API

The Dog API was integrated to display a random dog image.

API URL:

https://dog.ceo/api/breeds/image/random

Users can load a new image using the provided button.

## Disclaimer

A disclaimer was added to indicate that API-generated content is provided by third-party services and that I am not responsible for the generated content.

# Cookies / Client Storage

JavaScript local storage is used to remember previous visitors.

First-time visitors see:

"Welcome to my homepage for the first time!"

Returning visitors see:

"Welcome back! Your last visit was <date and time>"

The stored value is updated on each visit.
